## Docker

```bash
docker pull rebelstream/iptv:1.0.0
```

### Docker Compose

```yaml
services:
    iptv:
        image: rebelstream/iptv:1.0.0
        container_name: iptv
        ports:
            - "8080:8080"
        environment:
            TZ: "America/New_York"
        volumes:
            - ./data:/app/data
        restart: unless-stopped
```

### Docker Run

```bash
docker run -d \
  --name iptv \
  -p 8080:8080 \
  -e TZ=America/New_York \
  -v ./data:/app/data \
  --restart unless-stopped \
  rebelstream/iptv:1.0.0
```

### Updating

```bash
docker pull rebelstream/iptv:1.0.0
docker compose up -d
```

## Endpoints

| URL | Description |
|-----|-------------|
| `http://<server>:8080/playlist?key=YOUR_KEY` | M3U playlist |
| `http://<server>:8080/epg?key=YOUR_KEY` | EPG guide (XML) |
| `http://<server>:8080/epg.gz?key=YOUR_KEY` | EPG guide (gzip) |
| `http://<server>:8080` | Web dashboard |

> **Note:** The `?key=` parameter is only required if an API key has been generated in the web dashboard under **Settings**.
